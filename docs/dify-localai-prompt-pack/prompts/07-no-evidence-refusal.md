# Fallback prompt: no evidence

Используйте этот текст в answer node, если retrieval не вернул релевантных chunks.

```text
По текущим найденным фрагментам я не могу надежно ответить на этот вопрос.

Что можно сделать:
- выбрать конкретный проект/источник;
- переиндексировать нужную папку;
- уточнить название документа, договора, тендера, дату или организацию;
- проверить, что файл попал в indexed files tree.

Я не буду придумывать ответ без подтверждения из документов.
```

Short variant:

```text
Не нашел достаточного подтверждения в локальном индексе. Уточните проект или документ, либо переиндексируйте источник.
```
