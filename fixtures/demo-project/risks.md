# Риски демо-проекта

## Основные риски

Основные риски указаны как учебный список:

- задержка исходных материалов;
- изменение объема работ после согласования бюджета;
- недоступность площадки для проверки;
- перенос контрольной точки из-за уточнения требований.

## Меры контроля

Каждый риск проверяется еженедельно и фиксируется в журнале статуса.
